🌦️ Weather Predictor Simulation System

📌 Overview

The Weather Predictor Simulation System is a Python-based project that simulates multi-day weather forecasts using statistical probability and randomization techniques. It generates realistic weather conditions along with temperature, humidity, wind speed, weather advice, and extreme weather alerts.

This project was built to strengthen core Python programming skills while demonstrating the use of simulation-based forecasting.

---

✨ Features

- Multi-day weather forecasting
- Probability-based weather simulation
- Dynamic temperature generation
- Humidity and wind speed simulation
- Weather advice based on forecast
- Heatwave and strong wind alerts
- Weather summary report
- Clean and easy-to-read console output

---

🛠️ Technologies Used

- Python
- Random Module
- Loops
- Conditional Statements
- Lists
- Variables

🚀 How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

python weather_predictor.py

5. Enter the number of days to generate the weather forecast.
